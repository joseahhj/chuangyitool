# 创意工具

可以上传图片，编辑文字，拖拽缩放，删除等；



## 入门

### 先决条件
* node 环境



### 安装与运行


安装相关依赖
````
npm i
````
启动之前，需要先打包第三方文件库，即生成dll文件夹

```
npm run dll
```
运行
````
npm run dev

````
运行链接
[localhost:8081/picmaker/template.html]

联调
````
npm run debug

````
